InboxAI — Prototipo MVP

Pantalla principal

Cabecera

- Logo: InboxAI
- Nombre de la gestoría
- Usuario
- Botón: Conectar Gmail

Resumen

Mostrar cuatro indicadores:

- Correos pendientes
- Correos urgentes
- Correos procesados hoy
- Tiempo estimado ahorrado

Bandeja inteligente

Cada correo mostrará:

- Remitente
- Asunto
- Cliente identificado
- Categoría
- Prioridad
- Resumen generado por IA
- Fecha y hora

Categorías iniciales:

- Factura
- Nómina
- Impuesto
- Documentación
- Consulta
- Nuevo cliente
- Otro

Prioridades:

- Alta
- Media
- Baja

Vista detallada del correo

Al abrir un correo se mostrará:

Correo original

- Remitente
- Destinatario
- Asunto
- Contenido
- Archivos adjuntos

Análisis de IA

- Cliente identificado
- Categoría
- Prioridad
- Resumen
- Acción recomendada

Respuesta sugerida

Campo editable con el borrador generado por IA.

Botones:

- Editar
- Regenerar respuesta
- Enviar
- Marcar como pendiente
- Archivar

Flujo principal

1. El usuario conecta Gmail.
2. InboxAI obtiene los últimos correos.
3. La IA analiza cada correo.
4. El usuario entra en la bandeja.
5. Revisa el análisis.
6. Edita o acepta la respuesta.
7. Envía manualmente.
8. El correo queda marcado como procesado.

Restricciones del MVP

- No se enviarán respuestas automáticamente.
- No se analizarán correos antiguos.
- Solo se admitirá una cuenta de Gmail por gestoría.
- Los archivos adjuntos se mostrarán, pero no se procesarán en profundidad.
- No habrá aplicación móvil.
